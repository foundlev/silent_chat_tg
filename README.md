# Проект с внутренней экономикой

Интерактивный Телеграм чат со своей экономикой. Администрация отсутствует, всё решается через голосования. Включает механики исключения за неактивность, банковские счета с начислением, взломы, внутреннюю биржу, объединения в группы и другое. Завершился конкурсом с денежными призами для ТОПов.

### Используемые библиотеки:
- Aiogram
- Aiofiles
- Aiomysql
